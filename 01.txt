git   =>github    码云(git,svn) 
分布式管理系统 
三端：本地区，暂存区，远端区

命令：
    git init  项目初始化
    git status  查看文件状态
    git add <file> 添加单个文件
    git add.   添加所有
    git add <file><file><file>添加多个文件
    git commit -m + '注释'   提交到暂存区
    git remote add origin 地址 链接远程仓库
    git push -u origin master  第一次提交
    git push  后续提交
    git cinfig --global user.name+'用户名'
    git cinfig --global user.email+'邮箱'
